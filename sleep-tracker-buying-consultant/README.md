# Sleep Tracker Buying Consultant

> Turns any AI agent into an expert sleep tracker buying consultant.

## What it does

Guides sleep tracker buyers through a structured consultation to identify exactly which
specs they need for their specific situation, health goals, wearing comfort, smartphone
ecosystem, and region — without relying on biased sales advice. Delivers a prioritised
spec list and up to 5 matched product suggestions.

## How it works

1. Agent interviews the user with targeted, research-backed questions (health goals, form
   factor comfort, smartphone OS, bed-sharing situation, battery needs, privacy preferences,
   subscription tolerance, region)
2. Applies a sensor-needs decision matrix and clinical sleep benchmarks to determine required
   specs — and flags when a clinical consultation should accompany the device purchase
3. Delivers a structured spec recommendation: non-negotiable → recommended → optional
4. Suggests up to 5 real products matching the user's confirmed specs, tailored to their region

## Requirements

- No external APIs or environment variables required
- No runtime dependencies
- Works with any AI agent that supports SKILL.md (OpenClaw, ClawHub, etc.)
- Pure instruction-based — agent reasoning does the work

## Installation

Add via ClawHub or reference the SKILL.md directly in your agent configuration.
